# Release Notes

## v2.0.0 - 2026-04-26

### 🎉 重大更新

#### 🌟 项目更名
- 项目名称从"个人知识库"更名为"智慧管理中心"
- 更准确地反映了系统的综合管理特性

### ✨ 新增功能

#### 🍋 猕猴桃销售管理模块
1. **订单管理**
   - 支持创建、编辑、删除销售订单
   - 客户名称、运单号码管理
   - 规格选择（5斤装/10斤装）

2. **数据字段**
   - 数量管理（整数类型）
   - 支付金额（保留2位小数）
   - 发货状态（已发货/未发货）

3. **实用功能**
   - 导出订单到Excel（CSV格式）
   - 批量删除订单
   - 订单选择导出后自动清除选择并隐藏删除按钮

4. **报表统计**
   - 按客户分组统计
   - 按规格汇总统计
   - 总数量和总金额统计
   - 分页显示，每页10个客户
   - 实时数据同步，订单变更后自动更新报表

#### 📚 知识库模块优化
- 保留所有原有知识库功能
- Markdown编辑器、代码高亮
- 用户认证、分类管理、收藏功能
- 搜索、浏览统计、响应式设计

### 🔧 技术优化

#### 前端优化
1. **布局调整**
   - 优化销售订单列表布局
   - 防止运单号码折行
   - 报表列宽优化，数量列固定80px防止被挤压
   - 金额列自动调整宽度，支持大金额显示

2. **UI/UX改进**
   - 统一表头字体大小为15px
   - 优化报表分页控件
   - 改善按钮点击反馈

#### 后端优化
1. **API增强**
   - 新增销售订单管理API
   - 新增销售报表统计API
   - 支持分页查询

2. **数据库**
   - 自动数据库迁移
   - 新增字段自动添加

### 🐛 问题修复

- 修复订单编辑后数据不同步问题
- 修复删除后删除按钮不自动隐藏问题
- 修复多选功能被意外移除的问题
- 修复备注列不显示内容的问题
- 修复数据库路径错误问题

### 📦 项目配置

- 更新启动信息为"智慧管理中心"
- 更新README.md完整文档
- 项目结构优化整理

### 🚀 使用说明

1. **启动服务**
   ```bash
   python app.py
   # 或
   start.bat
   ```

2. **访问地址**
   - 本地：http://localhost:5001
   - 内网：http://[你的IP]:5001

3. **功能导航**
   - 左侧菜单切换知识库/销售模块
   - 销售模块下有"销售订单"和"销售报表"两个子菜单

### 📝 变更记录

#### 数据库变更
- kiwi_sales表新增字段：
  - remark (TEXT) - 规格备注
  - quantity (INTEGER) - 数量
  - payment_amount (REAL) - 支付金额
  - status (TEXT) - 发货状态

#### 文件变更
- app.py - 新增销售模块API
- static/index.html - 新增销售模块界面
- README.md - 更新完整文档
- 新增：RELEASE_NOTES.md - 发布说明
